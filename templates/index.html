Welcome to the Cafe & Wifi API

This project is about designing our own RESTful APIs that can perform:

  • GET method
  • POST method
  • PATCH method
  • DELETE method

This API provides information about cafes and their wifi availability.
The methods were tested on POSTman API. Check the documentation to know more on how to use our API.

Read the Documentation

We know a lot of public APIs. In many cases, APIs allow us to access website data or services. Many companies provide APIs for developers, offering valuable data such as Bitcoin prices and restaurant reviews.

What if we had access to useful data that others might need? For example, if we collected information on all the cafes in a city and identified which ones were best for remote work, we could create an API. This project was about buliding a full REST API from scratch using Flask.
